# RedLine

**RedLine is a Chrome extension that rewrites headlines and blurbs on web pages into various ideological perspectives instantly. It aims to help users understand how language can be used to frame information from different viewpoints.**

## Features

*   Rewrites text content on web pages (specifically targeting H1-H6, P, and LI elements).
*   Offers a wide range of ideological perspectives for rewriting, including: Liberal, Conservative, Radical Black Left, Neoliberal, Techno-utopian, Socialist, Christian Nationalist, Zionist, Anti-Zionist, Accelerationist, Anarchist, Populist Left, Populist Right, Conspiracy-core, Climate Doomer, Pan-Africanist, Feminist, Libertarian, Eco-socialist, Transhumanist.
*   Option to simplify the language of the rewritten content.
*   Provides visual feedback on the page while processing text.
*   User-friendly popup interface to select perspectives and options.

## How it Works

1.  The extension is activated via its popup menu.
2.  The `popup.js` script captures user selections (perspective, simplification).
3.  When "Rewrite Page" is clicked, `content.js` is triggered on the active tab.
4.  `content.js` identifies relevant text elements, excluding common navigational and footer areas.
5.  Each piece of text is sent to `background.js`.
6.  `background.js` constructs a prompt using the selected perspective and text, then calls the DeepSeek AI API (`deepseek-chat` model) to get the rewritten version.
7.  The rewritten text is then displayed on the page.

## Security Warning

**Important Security Notice:** The current version of this extension (`Chamber-main/chamber_corrected_final/background.js`) contains a hardcoded API key for the DeepSeek AI service. This is a significant security risk. **Do not use the extension in production or share it widely with the current key.** You should replace it with your own API key and ideally implement a secure key management solution (e.g., requiring users to enter their own key via an options page) before any serious use.

## Installation and Setup

1.  Download or clone this repository to your local machine.
2.  Open Google Chrome and navigate to `chrome://extensions`.
3.  Enable "Developer mode" using the toggle switch in the top right corner.
4.  Click the "Load unpacked" button.
5.  Select the `Chamber-main/chamber_corrected_final` directory from the cloned repository.
6.  **Crucial Step:** Before extensive use, open `Chamber-main/chamber_corrected_final/background.js` and replace the placeholder `DEEPSEEK_API_KEY` with your own valid DeepSeek API key.

## How to Use

1.  Once installed, click on the RedLine extension icon in your Chrome toolbar.
2.  The popup will appear.
3.  Select your desired ideological perspective from the dropdown menu.
4.  Check the "Simplified Language" box if you want the output to be easier to read.
5.  Click the "Rewrite Page" button.
6.  The content on the current web page will be updated according to your selections. Status messages ("Processing...", "Done!", "Error...") will be shown in the popup.

## File Structure Overview

*   `manifest.json`: Defines the extension's metadata, permissions, and core components (background script, content scripts, popup).
*   `popup.html` / `popup.js` / `popup.css`: Manage the extension's user interface for selecting perspectives and initiating rewrites.
*   `content.js`: Injected into web pages to identify text elements and replace them with rewritten versions.
*   `styles/contentStyles.css`: Basic styling for visual feedback during the rewriting process on the web page.
*   `background.js`: Acts as the service worker, primarily responsible for communicating with the DeepSeek AI API to get rewritten text based on prompts. **(Contains the API key)**.

## Disclaimer

The rewrites are generated by an AI model and may not perfectly represent the nuances of each ideological perspective. Use this tool critically as a way to explore language and framing, not as a definitive source of ideological truth.
